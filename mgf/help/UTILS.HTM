Содержание / Utils

Углы
float angleNormal(float a);
Обрезает угол a так, чтобы он оставался в диапазоне (0.f; 360.f).
float angleApproach(float val, float a1, float a2);
Приближает угол a1 к углу a2 на значение val с учётом нормализации и специфики углов. Возвращает результирующий угол.
содержание / вверх
Числа с плавающей точкой
float floatApproach(float val, float f1, float f2);
Приближает f1 к f2 на значение val. val - всегда положительное число. Возвращает результирующее число с плавающей точкой.
