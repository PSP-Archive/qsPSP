MGFVERTEX
struct MGFVERTEX;
Структура вершины (вертекса) полигона.
Параметры
float u, v; - текстурные кординаты.
unsigned int color; - цвет вершины
float x, y, z; - координаты в пространстве.

MGFQUAD
Методы
void Reset(void);
Выставляет значения по умолчанию для всех параметров квада. Вызывается в конструкторе.
void SetVertex(float width, float height, bool center = true);
Устанавливает пространственные координаты вертексов квада. Для центрирования квада center должен равняться true.
void SetTexCoord(float x, float y, float width, float height);
Устанавливает текстурные координаты вертексов квада. Указывается в диапазоне (0.f; 1.f).
void SetColor(int color);
Устанавливает цвет вертексов квада.

Параметры
float Width, Height; - ширина и высота квада.
MGFVERTEX v[4]; - 4 вертекса, образующие квад.
